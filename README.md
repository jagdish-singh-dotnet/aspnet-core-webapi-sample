# ASP.NET Core Web API Sample

This repository demonstrates a clean and scalable ASP.NET Core Web API
following industry best practices.

## Tech Stack
- ASP.NET Core
- Web API
- Entity Framework Core
- SQL Server
- JWT Authentication

## Features
- RESTful API design
- JWT-based authentication
- Layered architecture
- Dependency Injection
- Clean and maintainable code structure

## Purpose
This project is created for learning, demonstration, and interview discussions.
It does not contain any proprietary or production data.
